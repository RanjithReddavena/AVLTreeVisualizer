# AVL Tree Visualizer

This project demonstrates the working of a Self-Balancing Binary Search Tree (AVL Tree) along with a visualization of insertion and balancing operations.

## Features

- AVL Tree insertion
- Automatic tree balancing
- Visualization of nodes
- Educational explanation of AVL rotations

## Technologies Used

- HTML
- CSS
- JavaScript

## What is an AVL Tree?

An AVL Tree is a self-balancing Binary Search Tree where the height difference between left and right subtree (Balance Factor) is at most 1.

Balance Factor = Height(left subtree) - Height(right subtree)

If the balance factor becomes greater than 1 or less than -1, rotations are performed.

## Rotations in AVL Tree

1. Left Left (LL) Rotation
2. Right Right (RR) Rotation
3. Left Right (LR) Rotation
4. Right Left (RL) Rotation

## Time Complexity

| Operation | Complexity |
|----------|------------|
Search | O(log n)
Insert | O(log n)
Delete | O(log n)

1. Clone the repository
